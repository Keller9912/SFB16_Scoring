- Play-by-play: [nflverse](https://nflverse.com) · 2021–2025 regular season · Weeks 1–17
- Rosters: nflverse season rosters with gsis_id linking

## Scoring Rules (SFB16)
| Category | Points |
|---|---|
| Touchdown | 6 |
| 2PT Conversion | 2 |
| Per Passing Yard | 0.04 |
| Per Rush/Rec Yard | 0.1 |
| Per Reception | 0.5 |
| Per First Down (Rush/Rec) | 0.5 |
| TE Bonus per Reception | +1 |
| TE Bonus per First Down | +1 |
| 300+ Yard Passing Game | +10 |
| 400+ Yard Passing Game | +10 |
| 100+ Yard Scrimmage Game | +10 |
| 200+ Yard Scrimmage Game | +10 |
| 40+ Yard Passing Play | +10 |
| 40+ Yard Rushing Play | +10 |
| 20+ Yard Receiving Play | +10 |
